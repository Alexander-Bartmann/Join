# 🧩 Join – Task Manager (Teamprojekt)

Ein moderner **Task Manager**, inspiriert vom **Kanban-System**.  
Erstelle, organisiere und verwalte Aufgaben mit Drag & Drop, Benutzerzuweisung und Kategorien.

🌐 **Live-Version:** [alexander-bartmann.de/Join](https://alexander-bartmann.de/Join/)

---

## 🚀 Tech Stack

- **HTML / CSS / JavaScript (ES6)**
- **Modulares Projektsetup**
- **Drag & Drop API**
- **Responsive Design**

---

## 📄 Projektbeschreibung

> “Task Manager inspiriert vom Kanban System.  
> Erstelle und organisiere Aufgaben, nutze Drag & Drop, weise Benutzer und Kategorien zu.”

Das Projekt wurde in einem **2er-Team** innerhalb von **8 Wochen** entwickelt.  
Die Organisation erfolgte über **Trello**, wodurch der Entwicklungsprozess transparent und effizient war.

---

## 💡 Arbeitsprozess

- Aufgabenplanung & Tracking über **Trello**
- Aufgabenstatus in Spalten wie *To Do*, *In Progress*, *Done*
- Teamkommunikation über regelmäßige Reviews
- Gleichmäßige Arbeitsteilung im Zweierteam

---

## ✨ Features

- Aufgaben erstellen, bearbeiten, löschen
- Drag & Drop zwischen Spalten
- Benutzer- und Kategorienverwaltung
- Responsive UI für Desktop & Mobile

---

## 🧠 Meine Erfahrung

> “Ich habe in einem Zweierteam gearbeitet. Wir haben die Arbeit gleichmäßig aufgeteilt und regelmäßig kommuniziert.”

Dieses Projekt hat meine **Teamfähigkeit** und mein **Projektmanagementverständnis** gestärkt.
